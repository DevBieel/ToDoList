Boas vindas usuário(a)

Meu nome é Gabriel, sou estudante de Análise e Desenvolvimento de sistemas e esse é meu primeiro projeto FullStack, uma ToDo List.

Nesse meu projeto, utilizei o framework Django da linguagem de programação Python junto com HTML, CSS/Sass e JavaScript. É meu primeiro projeto feito com Django e estou muito feliz com o resultado.

Esse meu projeto foi inspirado no projeto do curso de Django do Hora de Codar mas com algumas modificações.

Nessa minha ToDo List é possível adicionar tarefas, editar ou excluir as tarefas e também buscar as tarefas pelo nome. Usei também o Paginator do Django para criar o sistema de paginação das tarefas.

Criei dois tópicos abaixo falando sobre o sistema de autenticação usado no projeto e sobre a conexão do projeto com o banco de dados:

Autenticação

Para a autenticação de usuários utilizei o Django auth com algumas modificações.
No Django auth padrão o modelo de autenticação é com username e senha, mas no meu projeto fiz uma modificação oara que o modelo de login seja feito com e-mail e senha.
Decidi implementar um login nessa ToDo List para que cada usuário para que as tarefas sejam atribuidas a um usuário específico, e também para treinar o Django auth.

Banco de Dados

O SGBD que eu utilizei nesse projeto foi o MySQL. Usei o pacote mysql-connector-python para fazer a conexão do Django com o banco de dados.
Para a criação das tabelas usei o brModelo para criar o DER(diagrama entidade relacionamento), para definir quais colunas seriam chave primária, chave estrangeira e etc.
